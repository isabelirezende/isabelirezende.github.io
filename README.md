## 📌 Sobre o projeto

O objetivo deste projeto foi criar uma página web utilizando o framework **Bootstrap**, aplicando conceitos como:

* Grid System (layout responsivo)
* Componentes (navbar, panels, botões)
* Estilização com CSS
* Estruturação semântica em HTML

Além disso, a página inclui:

* Informações profissionais
* Formação acadêmica
* Projetos desenvolvidos
* Competências técnicas
* Objetivo profissional para os próximos 5 anos

---

## 🛠️ Tecnologias utilizadas

* HTML5

* Bootstrap 3
* JavaScript (básico para interação)

## 🎯 Funcionalidades
